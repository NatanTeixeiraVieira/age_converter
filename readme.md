<h1>Age Converter</h1>

<p>Este projeto converte uma data de nascimento em idade, onde o resultado pode ser em um dos seguintes:</p>
<ul>
    <li>Anos, meses e dias;</li>
    <li>Meses e dias;</li>
    <li>Dias.</li>
</ul>
